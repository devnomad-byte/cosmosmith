# Cosmosmith Market Research

## Purpose

Turn current external evidence into `docs/cosmosmith/research-brief.md`.

## Workflow

1. Read `docs/cosmosmith/idea-brief.md`, `proposal.md` if present, and `task.md`.
2. Search the web for comparable products, user expectations, pricing, standards, failures, and constraints.
3. Prefer official, primary, or high-quality sources. Label community examples as examples.
4. Separate facts from inferences.
5. Write or update `docs/cosmosmith/research-brief.md`.
6. Update `task.md` when research changes product direction, architecture, UI, QA, or release tasks.

## Output Requirements

- Source links.
- Competitor or analogue table.
- Facts.
- Inferences.
- Product implications.
- Design implications.
- Architecture implications.
- Verification or release implications.

## Completion

Stop when the next role can make decisions from evidence rather than guesses.
